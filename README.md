-добавление в конец<br>
-удаление по индексу<br>
-проверка наличия по значению<br>
-проверка пустоты списка<br>
-получение по индексу<br>
-добавление в конкретную позицию<br>

-постановка в очередь<br>
-получение первого в очереди<br>
-проверка пустоты<br>

-get(key, value)<br>
-containsKey(key)<br>
-isEmpty()<br>
